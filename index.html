Mengenal dasar TAG HTML

Tag HTML adalah penanda awal dan akhir elemen HTML, yang ditulis dengan kurung siku (<â€¦>). Tag HTML berfungsi sebagai kerangka dasar yang mendefinisikan struktur dan isi halaman web.

Jenis Tag HTML :

  • Tag pembuka, ditandai dengan "<"
  • Tag penutup, ditandai dengan "< /"

Contoh Tag HTML :

  • < html > untuk menunjukkan pembuka HTML
  • < head > untuk bagian kepala dari dokumen
  • < title > untuk judul web
  • < body > untuk bagian body dari dokumen
  • < h1 > untuk menunjukkan heading
  • < p > untuk sebuah paragraf
  • < img > untuk gambar

Aturan Penulisan Tag HTML :

  • Setiap tag HTML harus memiliki tanda pembuka (< >) dan tanda penutup (< / >) yang sesuai
  • Setiap tag HTML harus ditulis dalam huruf kecil
  • Setiap tag HTML harus memiliki nama tag yang benar
  • Pastikan menutup tag dengan benar, terutama jika tag HTML bertumpuk-tumpuk

Fungsi Tag HTML :

  • Tag HTML format mengatur tampilan teks dan elemen lain di halaman web
  • Tag HTML juga berguna sebagai jembatan dengan CSS dan JavaScript
  • Kita bisa menambahkan gaya dan fungsi interaktif ke halaman web

Mengenal Paragraf dasar HTML

Dasar paragraf dalam HTML adalah tag < p > yang digunakan untuk membungkus teks menjadi paragraf. Paragraf HTML adalah elemen tingkat blok yang membuat baris baru dan mengambil lebar penuh dari wadah induknya.

Sintaks dasar paragraf HTML :

  • Tag pembuka < p > menunjukkan awal paragraf.
  • Tag penutup < / p > menunjukkan akhir paragraf
  • Konten apa pun yang ditempatkan di antara tag pembuka dan penutup akan dianggap sebagai bagian dari paragraf

Ciri-ciri paragraf HTML :

  • Paragraf selalu dimulai pada baris baru
  • Browser secara otomatis menambahkan ruang kosong (margin) di sekelilingnya
  • Paragraf HTML dapat berupa pengelompokan struktural konten terkait, seperti gambar atau kolom formulir

Mengenal Heading dasar dalam HTML

Heading dalam HTML adalah tag yang digunakan untuk membuat judul dan subjudul pada halaman web. Heading terdiri dari enam tingkatan, yaitu < h1 >, < h2 >, < h3 >, < h4 >, < h5 >, dan < h6 >.

Fungsi heading Menunjukkan struktur hierarki konten, Memudahkan pengguna dan mesin pencari memahami struktur halaman, Memudahkan pembaca menemukan topik yang dicari, Meningkatkan keterbacaan halaman, Penting dalam kaitannya dengan teknik Search Engine Optimization (SEO).

Cara menggunakan heading :

  • Gunakan H1 sebagai judul utama pada halaman
  • Gunakan H2 sebagai subjudul untuk subbagian
  • Gunakan H2-H6 subheadings untuk menyusun konten lebih lanjut
  • Jangan lewatkan level heading
  • Hindari mengisi heading dengan kata kunci secara berlebihan
  • Pastikan tag H1 merangkum konten halaman dengan benar

MDN Web Docs menjelaskan bahwa secara default, semua elemen judul membuat kotak tingkat blok dalam tata letak.
